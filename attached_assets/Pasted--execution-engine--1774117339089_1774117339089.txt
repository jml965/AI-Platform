المطلوب: تحسين سلوك البحث والتنفيذ للوكيل execution_engine بشكل نهائي

========================================

1. قاعدة أساسية
   ========================================

إذا DOM يحتوي نص:
→ افترض أنه موجود في الكود
→ لا تفترض DB مباشرة

========================================
2) ترتيب البحث الإجباري (Search Strategy)
=========================================

الترتيب يجب أن يكون:

1. البحث بالنص (text)
2. البحث بـ className
3. البحث بـ data-testid
4. البحث بـ aria-label
5. البحث في ملفات Layout/Sidebar/Header/Nav
6. البحث في i18n
7. أخيراً DB (إذا فشل كل ما سبق)

========================================
3) البحث الذكي
==============

search_text يجب أن يبحث عن:

* النص المباشر
* t("...")
* label / title / name
* children داخل JSX

========================================
4) استخدام DOM path
===================

إذا DOM يحتوي:

class:
flex items-center

→ search_text عن:

"flex items-center"

========================================
5) auto-read (إجباري)
=====================

إذا search_text وجد match:

→ read_file مباشرة
→ لا search جديد

========================================
6) دعم attributes الحديثة
=========================

search_text يشمل:

* className
* data-testid
* aria-label

========================================
7) رفع حد العمليات
==================

MAX_ACTIONS_WITHOUT_EDIT = 8

لكن:

إذا:

* 3 search بدون نتيجة

→ توقف واطلب توجيه

========================================
8) fallback ذكي
===============

إذا لم يتم العثور:

ابحث مباشرة في:

* Layout.tsx
* Sidebar.tsx
* Header.tsx
* Navigation.tsx

========================================
9) متى تستخدم DB؟
=================

فقط إذا:

* النص dynamic
* مرتبط user / project / API

========================================
10) منع الاستنتاج الخاطئ
========================

ممنوع قول:

"النص غير موجود في الكود"

إلا بعد:

* 3 search (text + class + component)

========================================
11) المسار النهائي
==================

DOM
→ search (text)
→ search (class)
→ read_file
→ edit_component

========================================
ابدأ التنفيذ الآن
